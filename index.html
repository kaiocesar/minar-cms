Bem-vindo ao projeto MINAR.

O projeto MINAR é desenvolvido inteiramente com JavaScript. Tendo como base o MongoDB, Express, AngularJS e NodeJS, formando assim o conjunto chamado MEAN. O objetivo é criar um sistema independente das velhas linguagens ainda tão disseminadas, podendo assim aproximar aqueles que estão começando a programar, ou que ainda não estão familiarizados com NodeJS.

$ git clone https://github.com/Hunteros/MINAR.git
$ npm start

Desenvolvimento

Devido o objetivo didático do projeto, a principais partes do projeto estaram comentadas, explicando cada trecho do código. Caso esteja faltando algo, sinta-se a vontade em agregar seus conhecimentos ao projeto.
